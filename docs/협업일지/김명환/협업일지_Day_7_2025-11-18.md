## [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?
**답변**:
```
[인사이트]
- 네이버 웨일 브라우저를 사용하면 HWP 파일을 간편하게 PDF로 변환할 수 있음을 발견했습니다.
- 데이터포탈 API를 통해 제안서 데이터를 성공적으로 수집할 수 있었습니다.

[문제점]
- 웨일 브라우저와 크롬 드라이버의 버전 불일치로 인해 자동화에 실패했습니다.
- 크롬 드라이버 버전 140을 테스트해야 하는 추가 작업이 필요합니다.
```

## [6] 일정 지연이나 협업 중 어려웠던 점이 있다면?
**답변**:
```
브라우저 자동화를 위한 크롬 드라이버 설정에서 버전 호환성 문제가 발생하여
예상보다 시간이 더 소요되었습니다. 웨일 브라우저 전용 드라이버 또는
최신 버전의 크롬 드라이버(v140)를 테스트해야 하는 상황입니다.
```
