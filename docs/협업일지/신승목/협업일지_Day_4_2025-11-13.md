## [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?
**답변**:  
```
HWP 파일 형식이 모두 OLE2 Compound File (HWP 5.x binary)로 동일한 것을 확인
docx 변환과 pdf 변환이 있는데 docx는 표 처리에 강하지만 db에 저장할 때 전체 페이지를 1로 저장
pdf 변환은 페이지 구분이 확실해서 페이지 정보를 잡아낼 수 있지만 표가 페이지 경계로 분할되면 제대로 처리되지 못할 가능성 있음
우선 pdf 변환을 해서 어떻게 저장되는지 확인해 볼 계획
```
