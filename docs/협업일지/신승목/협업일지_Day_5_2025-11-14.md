## [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?
**답변**:  
```
PDF를 마크다운으로 변환해서 DB에 저장된 텍스트 내용을 살펴봤을 떄, 영어는 띄어쓰기가 잘 되어 있는데, 우리말이 쉼표를 제외하고는 띄어쓰기가 되어 있는 것을 찾기 힘들 정도로 되어 있었다.
표의 경우, 마크다운이 잘 지켜졌고, 여러 셀에 걸쳐져 있는 셀은 해당 셀의 내용이 각 줄마다 반영되어 있었다.
모든 문서에서 표지와 목차가 처음에 들어가기 때문에 문서 내에서 인식하고 있는 페이지 번호와 마크다운으로 저장하는 페이지 번호가 다르다. 예를 들면, 
- 4
--- 페이지 7 ---
이런 형식으로, 우리가 파일을 볼 때는 페이지 번호가 7이지만, 문서 상에는 4페이지인 경우 어떻게 처리할 지 논의가 필요하다.
```
