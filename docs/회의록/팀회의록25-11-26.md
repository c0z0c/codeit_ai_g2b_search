## 주요 안건 및 논의사항

### 1. 발표 자료 준비 현황
**발표 자료 작성 일정**:
- 각자 파트별 내용 검토 및 수정
- 16시부터 김명환님이 내용 취합 후 PPT 작성
- 발표 자료를 발표자료 폴더에 업로드
- 사이트를 공개 모드로 전환 예정
- 링크 또는 파일 형태로 제출 가능 (링크 권장 - 수정 용이)

**발표 자료 구성**:
- 종합보고서의 핵심 내용 활용
- AI 도구(캔들 등) 활용하여 간결하게 15장 정도로 구성
- 연구 보고서 링크 포함하여 별도 자료 존재 명시

### 2. 데모 동영상 제작
**담당자**: 오형주
**마감**: 11/26 22:00 (편집 필요시), 자기 전까지 완성
**길이**: 1분 ~ 1분 30초 내외
**내용 구성**:
- 일반 검색
- 심층 검색 (정상 검색 + 오류 검색)
- 게시물 검색
- 평가 기능
- 히스토리 기능 (사이트 나갔다가 돌아와도 유지)
- 데이터 포털 연동
**편집**: 시간이 오래 걸리는 부분은 편집으로 빠르게 처리 또는 잘라내기
**업로드**: YouTube 업로드 (형주님 또는 명환님)
**활용**: 발표 초반에 시스템 동작 시연용

### 3. 완료 보고서 현황
**작성 상태**:
- 기본 보고서 작성 완료 및 공유됨
- 각자 파트별 수정 사항 반영 필요

**평가 지표 관련**:
- RAG 평가 로직은 테스트 완료
- 화면 표시 부분은 시간 부족시 향후 계획으로 이동 가능
- 3주 개발 기간 고려하여 유연하게 대응

### 4. 데이터 포털 업데이트
**담당자**: 오형주
**진행 상황**:
- 기존 데이터 처리 완료
- 시간 단위 업데이트 기능 구현
- 19일 14시~18시 업데이트 테스트 (약 1,000건)
- 어제 14시~16시: 3,000건 발생

**최적화 방안**:
- 점심시간(12시~14시) 업데이트 권장 - 공무원 점심시간 활용
- 금요일 점심시간으로 스케줄 조정 예정

**API 이해**:
- 공고 올라온 시점 기준 데이터 수집
- 실제 서비스 시 관련 문서 모두 수집 필요 (공지, 규격서 등)
- 데이터 포털 API 문서 참고 권장 (날씨, 환경 정보 등 활용 가능)
