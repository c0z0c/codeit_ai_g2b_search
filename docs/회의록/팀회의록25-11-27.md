## 주요 안건 및 논의사항

### 1. 시스템 마무리
- UI쪽 설정 마무리 작업
- 테스트 시 설정 환경을 원 상태로 복구

### 2. 보고서 검수
- 김명환님이 작성해주신 보고서 검수
- 작성된 보고서 검토 및 수정

### 3. 발표 자료 준비
**발표 자료 구성**:
- 종합보고서의 핵심 내용 활용
- 차별성이 드러나는 내용을 적극 강조
- AI 도구(감마 등) 활용하여 간결하게 15장 정도로 구성
- 연구 보고서 링크 포함하여 별도 자료 존재 명시

### 4. 데모 동영상 제작
**담당자**: 오형주
**마감**: 11/27 22:00 (편집 필요시), 자기 전까지 완성
**길이**: 1분 ~ 1분 30초 내외
**내용 구성**:
- 일반 검색
- 심층 검색 (정상 검색 + 오류 검색)
- 게시물 검색
- 평가 기능
- 히스토리 기능 (사이트 나갔다가 돌아와도 유지)
- 데이터 포털 연동
**편집**: 시간이 오래 걸리는 부분은 편집으로 빠르게 처리 또는 잘라내기
**업로드**: YouTube 업로드 (형주님 또는 명환님)
**활용**: 발표 초반에 시스템 동작 시연용

### 5. 핵심 논의사항
